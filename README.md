# Inventory-Management-Dashboard

Redux install: npm i react-redux @reduxjs/toolkit dotenv
npm i redux-persist
npm i prisma @prisma/client

Build and Deploy a Complete Full Stack Inventory Management Dashboard Application using NextJS, Redux, and Node. The backend is powered by Node.js, using Prisma as the ORM to facilitate database interactions.

## Tech Stack
 - Next JS
 - Tailwind
 - Redux Toolkit
 - Redux Toolkit Query
 - Material UI Data Grid
 - Node.js
 - Prisma
